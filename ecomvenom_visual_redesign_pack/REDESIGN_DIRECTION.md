### Typography direction

Investigate and choose one system only:

```text
Primary: Urbanist / Satoshi / Neue Montreal / Inter Tight
Display: Urbanist / Space Grotesk / Satoshi
Body: Inter / Satoshi / Neue Montreal
```

Required feel:

- rounded geometric
- premium editorial
- readable under motion
- strong numerical display
- not playful
- not default SaaS

### Iconography

Use:

- minimal line icons
- small geometric symbols
- folder/document icons
- arrows
- dots
- status pills
- circular controls

Avoid:

- colorful icon packs
- emoji UI
- generic SaaS icons
- heavy illustrative icons

### Materials

Use:

- frosted glass panels
- translucent cards
- blurred shadows
- volumetric green lighting
- soft black gradients
- subtle grain only if performance-safe
- depth layering
- document-stack visual metaphors
- bento cards

Avoid:

- random neon everywhere
- cheap gradient blobs
- noisy textures
- overused glassmorphism without hierarchy

## Homepage Architecture Direction

Create a new full homepage architecture from scratch.

Suggested structure:

1. Cinematic hero with visible primary CTA above fold
2. Spatial product/funnel metaphor panel
3. Founder/trust proof module integrated as a premium asset
4. Problem-to-system transition
5. Dropshipping / drop servicing mechanism section
6. Roadmap / curriculum as bento or document-stack system
7. Proof / results gate
8. Offer / application section
9. FAQ / objection handling
10. Final conversion section

Do not preserve the current section visuals.

## Motion Direction

Use:

- scroll-driven narrative
- parallax depth
- blur-to-sharp reveal
- glass-panel elevation
- document-stack fan movement
- sticky CTA safe zones
- reduced-motion support
- performance-safe transitions

Avoid:

- animation spam
- blocking video playback
- scroll hijacking that feels broken
- mobile 300vh empty sections
- heavy global canvas loops
