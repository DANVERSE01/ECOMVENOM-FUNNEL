# MCP Toolbelt

Use this power whenever the user asks for MCP, Git inspection, web fetching, browser automation, repo status, page testing, screenshots, or Playwright checks.

## Required behavior

- Use git MCP for repository status, branches, diffs, and file inspection.
- Use fetch MCP for fetching public URLs and summarizing page content.
- Use playwright MCP for opening pages, checking titles, browser inspection, and UI testing.
- Always return PASS/FAIL with exact errors when testing MCP connectivity.
